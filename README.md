# README #

## Overview ##
Phalcon / AngularJS skeleton

Stack: Phalcon, AngularJS, Bootstrap, Bower, CoffeeScript, SASS, Grunt, Karma/Jasmine

## Installation steps ##
```
#!bash

git clone https://bitbucket.org/vano468/phangular.git
cd path/to/project
npm install
bower install
grunt build
mkdir app/cache
```
